# EWCircleView
<h3>swift轮转动画</h3>

<h5>实现效果:</h5>

静止时:子view对称排列,允许动态添加,0~24个都能较好的显示.

旋转时:中心view不动,子view随手势旋转,最下方子view变大突出.
